# Eclipse-Eclipse Java IDE Cheat Sheet

## General
Show available shortcuts    |CMD + shift + L

## Files
Open a resource i.e *.java    |CMD + shift + r
Save a file                   |CMD + s
Close a file                  |CMD + w
Cylce Editors Next            |CMD-F6
Cylce Editors Previous        |shift-CMD-F6

## Search
Search dialog                 |ctrl + H

## Testing
Run a unit test               |CMD + alt + x then T
